### 提交前检查项

- [ ] 网站可正常访问，HTTPS 有效
- [ ] 已有 ICP 备案（国内导航站强烈建议，部分站硬性要求）
- [ ] 首页已添加目标导航站的友情链接（多数站要求互链）

## 七、建议执行顺序（4 周节奏）

### 第 1 周：基础设施

1. 完善 speedce.com 中文页 / 帮助文档 / 关于我们
2. 注册百度、Google、Bing 站长平台并提交 sitemap
3. 在首页添加友链区块（预留 5–8 个导航站互链位）

### 第 2 周：核心导航提交

按顺序提交：

1. IDC 导航 → IDC 狐狸 → 桃子百科 → 霂明导航
2. itotii 导航 → 老吴站长导航 → 三栖导航
3. 芦虎导航（先挂友链再登录提交）→ 无峰导航（登录后提交）

### 第 3 周：内容与社区

1. 发布 2–3 篇站内 SEO 长尾文章（见 SEO 提纲文档）
2. 在 HostLoc 发体验帖（见 HostLoc 提纲文档）
3. 邮件联系 VPS 测评网、hi-linux 投稿

### 第 4 周：友链交换

参考 boce 友链列表，联系以下类型站点互换友链：

- DNS 解析服务商（如免费智能 DNS、域名解析类）
- VPS/主机测评站（主机测评网、VPS 小学生等）
- 站长 SEO 工具站（桔子 SEO、聚查网等）

## 九、常见拒审原因与对策

| 拒审原因 | 对策 |
|----------|------|
| 未添加友链 | 提交前在首页醒目位置添加对方链接，截图留存 |
| 无 ICP 备案 | 先完成备案再申请国内导航；海外导航可尝试 IID.HK 等 |
| 内容太少 / 新站 | 补充帮助文档、关于页、至少 3 篇 SEO 文章后再提交 |
| 仅有英文界面 | 增加中文标题、描述、操作说明 |
| 建站时间太短 | 霂明导航等会以此拒审；先完善内容，1–2 个月后再申 |
| 与现有收录站重复 | 强调差异化：地图可视化、国内+全球双视图、简洁无广告 |
