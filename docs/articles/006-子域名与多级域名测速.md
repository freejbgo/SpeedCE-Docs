# 域名、子域名、多级域名测速有什么不同

> 关键词：子域名测速,域名检测,API测速,多级域名  
> 分类：基础知识  
> 工具： [SpeedCE](https://www.speedce.com) | [中文版](https://speedce.com/?lang=zh-CN)

主站 www.example.com 正常，不代表 api.example.com、cdn.example.com、m.example.com 都正常。子域往往指向不同服务器、不同证书、不同 CDN 配置，需要**独立检测**。

SpeedCE 输入框支持主域名、子域名、多级域名（如 a.b.example.com）以及裸 IP。每次只需切换输入目标，协议仍可选 HTTP、HTTPS 或 PING。

**常见疏漏：**
- 主站证书含 www，但 API 子域证书遗漏；
- CDN 只加速 www，API 直连源站却防火墙未放行；
- 移动端 m. 子域 DNS 未同步修改。

建议维护一份「对外子域清单」，每月用 SpeedCE HTTPS + 中国节点逐条过一遍。地图全绿再归档，异常立即跟进。
