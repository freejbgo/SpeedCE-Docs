> 关键词：迁机测速,DNS生效,服务器迁移  
> 分类：场景实战

迁机后最大坑：你以为 DNS 改了，部分省份仍解析旧 IP。用 SpeedCE HTTPS 测**域名**（非 IP），中国节点每 10 分钟测一次，观察异常点是否减少。

全国仍红 → 新服务器 Web/证书/防火墙问题；仅零星红且递减 → DNS 缓存，继续等待 TTL；固定某省持续红 → 区域性 DNS 问题。

迁机前调低 TTL，迁后坚持用地图验收。https://speedce.com/?lang=zh-CN
