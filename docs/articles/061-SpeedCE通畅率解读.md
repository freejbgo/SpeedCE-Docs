SpeedCE 测速完成后，顶部显示：检测节点数、通畅、异常、已跳过、平均延迟。

**通畅** = 成功访问的节点数；**异常** = 超时或失败；**已跳过** = 调度未参与的节点，可忽略。通畅率 = 通畅/(通畅+异常)。若低于 90%，应警惕；低于 70%，基本属故障。
