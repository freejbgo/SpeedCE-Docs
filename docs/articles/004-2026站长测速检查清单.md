# 2026 年站长必备的在线测速检查清单

> 关键词：站长工具,测速清单,网站巡检,运维检查

网站上线不是终点，而是需要持续巡检的起点。以下清单适用于个人博客、企业官网、API 服务，建议配合 SpeedCE 等多节点工具执行。

**变更后必测（每次）：** 改 DNS、换服务器、上 CDN、续签 SSL 证书、调整防火墙/Nginx 后，用 HTTPS + 中国节点测主域名。

**每周巡检：** HTTPS 测主域，看通畅率是否下降；抽查一个关键子域（如 api.）。

**每月体检：** 电信、联通、移动分别测一次；全球节点测目标市场国家；PING 测源站 IP（若未禁 Ping）。

**上线前验收：** 主域 + 子域 HTTPS 通畅率 ≥ 95%；证书有效期 > 30 天；地图无持续性区域全红。
