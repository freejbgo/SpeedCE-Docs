# Issue 标题

问：什么时候测域名、什么时候测 IP？

---

# Issue 正文

## 问题

迁机、买 VPS、怀疑 DNS 污染时，输入该填域名还是 IP？

## 回答

- **域名**：含 DNS、CDN、证书，模拟用户真实路径
- **IP**：绕过 DNS，验裸机或 VPS 线路

迁机后先测域名；买 VPS 先测 IP；域名与 IP 结果差异大时查 DNS。
