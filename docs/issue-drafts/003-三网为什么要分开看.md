# Issue 标题

问：电信正常、移动打不开，是服务器挂了吗？

---

# Issue 正文

## 问题

客服说「移动用户打不开」，自己用电信宽带访问却正常。要不要先重启服务器？

## 回答

**不一定。** 同一站点在电信、联通、移动下的路由可能完全不同。

排查建议：

1. 分别查看电信、联通、移动节点的结果（不要只看「全国平均」）
2. 若仅移动异常 → 查移动线路优化、跨网互联、CDN 移动节点
3. 若三网都红 → 再查服务器、证书、防火墙

## 案例

某站电信联通全绿、移动大片异常，扩容 CPU 无效，换带移动优化的 CDN 节点后恢复。问题在线路而非算力。
