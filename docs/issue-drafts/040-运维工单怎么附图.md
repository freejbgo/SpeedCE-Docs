# Issue 标题

问：回复用户「我这边正常」时，工单怎么写更有说服力？

---

# Issue 正文

## 问题

客服和开发扯皮，缺少第三方视角证据。

## 回答

模板示例：

> 我们刚用多省节点检测，电信/联通通畅率 96%，广东浙江北京正常。请告知您的省份与运营商，我们针对性复测。

附上**带时间戳的地图截图**（含协议与节点范围）。客观数据减少甩锅，比形容词有效。
