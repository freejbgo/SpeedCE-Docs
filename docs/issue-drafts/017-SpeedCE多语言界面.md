# Issue 正文（复制到描述框）

## 摘要

SpeedCE 内置国际化系统，右上角可切换语言，或通过 URL 参数访问：

- 简体中文：`?lang=zh-CN`
- English：默认
- 以及法语、越南语、泰语、俄语、日语、韩语、德语、西班牙语、葡萄牙语、阿拉伯语、印尼语、马来语、印地语等

中文界面标语：**覆盖中国各省市 · 全球节点 · 一键检测网络连通性**。海外团队成员可用英文界面操作同一套中国/全球节点，避免沟通成本。

出海团队、外籍运维、多语言客服场景均适用。官网：https://www.speedce.com。
