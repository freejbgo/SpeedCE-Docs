# Issue 标题

问：挂号系统可用性，为什么地图比平均延迟重要？

---

# Issue 正文

## 问题

医院 IT 关注平均响应时间，但投诉集中在某几个省。

## 回答

挂号场景是**零容错**：一个省持续超时 = 该省用户无法就医。

关注「异常节点比例」和分省地图，而非全国平均 ms。合规巡检可存档 HTTPS 全国节点截图。
