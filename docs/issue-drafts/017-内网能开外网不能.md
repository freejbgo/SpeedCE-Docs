# Issue 标题

问：公司内网能开、用户外网打不开，怎么证伪「我这边正常」？

---

# Issue 正文

## 问题

开发坚持本地能访问，客服持续收到外网投诉。

## 回答

内网能开常见原因：hosts、内网 DNS、VPN、浏览器缓存、走内网穿透。

**用外网多节点测速截图**作为工单附件，比口头争论有效。

四步法：HTTPS 国内节点 → 三网筛选 → 全球节点（若相关）→ HTTP/HTTPS 对比。

以外网地图为准达成修复共识。
