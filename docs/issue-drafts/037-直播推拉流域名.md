# Issue 标题

问：直播播放域和推流域要分别测吗？

---

# Issue 正文

## 问题

观众端卡顿，是播流域还是推流域问题？

## 回答

**分别 HTTPS/PING 测** `play.` 与 `push.`（或实际域名）。播流红、推流绿 → 查 CDN 播放链路；都红 → 源站或鉴权。不要混测一个域名代表全链路。
