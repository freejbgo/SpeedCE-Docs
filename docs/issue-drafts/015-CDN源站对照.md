# Issue 标题

问：接入 CDN 后变慢，先查 CDN 还是源站？

---

# Issue 正文

## 问题

上 CDN 后部分省份异常，不确定是 CDN 节点还是源站问题。

## 回答

做**两组对照**（同一时段）：

- A：加速域名（如 www，走 CDN）
- B：源站 IP 或 origin 子域（直连源）

| A | B | 结论 |
|---|---|------|
| 红 | 绿 | 找 CDN 配置/节点 |
| 绿 | 红 | 修源站 |
| 红 | 红 | 先源站 |

Cloudflare 橙云开关前后也可各测一次留档。
