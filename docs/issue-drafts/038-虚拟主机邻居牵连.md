# Issue 标题

案例：共享 IP 虚拟主机 sporadic 异常

---

# Issue 正文

## 现象

多节点 sporadic 红，本地一直正常，同一 IP 上其他站也异常。

## 可能原因

共享 IP 被牵连封禁或过载。

## 建议

对比邻居站点地图；考虑独立 IP 或换主机。单点本地测试无法发现此类问题。
