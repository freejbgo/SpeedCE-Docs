# Issue 标题

问：跨境独立站要测国内还是测海外？

---

# Issue 正文

## 问题

独立站面向欧美，只测国内节点够不够？

## 回答

不够。工作流建议：

1. 中国节点（运营、支付回调、国内客服）
2. 全球节点（目标市场：美欧东南亚）

两张地图决定是否在法兰克福、新加坡等加镜像或 CDN。
