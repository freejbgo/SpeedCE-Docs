# Issue 正文

## 问题

客户称某时段不可用，己方监控显示正常。

## 回答

多节点 HTTPS 截图（带时间）可证明「某时段某区域不可用」或「全网正常倾向用户侧网络」。非法律鉴定，但比单方日志更有公信力。重大变更前后各存档一次。
