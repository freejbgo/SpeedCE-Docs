# Issue 标题（复制到 GitHub New Issue）

SpeedCE 地图上「检测中」「等待」是什么意思？

# Issue 正文（复制到描述框）

## 摘要

地图节点四种状态：**通畅**（绿）、**异常**（红）、**检测中**（进行中）、**等待**（排队）。

测试过程中地图动态更新，不必等全部完成才判断——若已测节点大面积红，可先停测去修服务器。点击「停止测试」可提前结束。

进度条显示「正在检测节点 x/y」，透明可控。体验：https://speedce.com/?lang=zh-CN。
