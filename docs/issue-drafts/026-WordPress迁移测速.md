# Issue 正文（复制到描述框）

## 摘要

WordPress 迁到新主机后，除后台能登录外，务必做全国可达性检测。插件、主题、强制 HTTPS、缓存插件可能导致部分省份异常。

SpeedCE HTTPS 测首页域名，关注是否仅 HTTPS 异常（证书/Mixed Content）或全国超时（PHP-FPM/数据库）。迁移插件换域名后，用地图验证而非仅本地刷新。

https://speedce.com/?lang=zh-CN — WP 站长迁移必备一步。
