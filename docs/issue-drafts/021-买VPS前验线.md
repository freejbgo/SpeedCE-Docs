# Issue 标题

问：买 VPS 前如何验证商家线路是否吹牛？

---

# Issue 正文

## 问题

商家写「CN2 GIA」「三网直连」，到货前如何验证？

## 回答

拿到 IP 后：

1. PING + 国内多节点看延迟分布
2. 电信、联通、移动分别截图
3. 面向海外则补全球节点

电信联通 30–80ms、移动 150ms+ 常见于偏电信联通优化。不满意在退款期内换，地图数据可维权发帖。

验的是** IP**，买前可向商家要测试 IP。
