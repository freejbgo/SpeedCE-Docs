# Issue 标题

问：API 网关返回 502，如何先排除网络层？

---

# Issue 正文

## 问题

网关 502，不确定是后端挂了还是线路问题。

## 回答

1. HTTPS 测 API 域名全国节点
2. 若全国红 → 查网关 upstream、证书、LB 健康检查
3. 若仅某省红 → 更像线路或区域节点
4. LB 控制台「健康」但用户报错 → 健康检查路径可能与真实 API 不一致，外网地图对照

先地图定性，再下钻日志。
