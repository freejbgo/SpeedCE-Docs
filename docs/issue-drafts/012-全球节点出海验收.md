# Issue 标题

问：有海外用户时，国内测速正常就够了吗？

---

# Issue 正文

## 问题

国内地图全绿，但新加坡、美国用户反馈慢或打不开。

## 回答

不够。需要**全球节点**补测，并对照国内结果：

| 国内 | 海外 | 可能方向 |
|------|------|----------|
| 绿 | 红 | 海外 CDN/线路未覆盖 |
| 红 | 绿 | 国内备案、解析或线路 |
| 都红 | 都红 | 源站或域名级问题 |

出海业务建议国内、目标市场各测一次，截图留存。
