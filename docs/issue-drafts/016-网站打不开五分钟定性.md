# Issue 标题

问：用户说网站打不开，五分钟如何定性故障范围？

---

# Issue 正文

## 问题

「我这边正常」和「打不开」僵持时，外网第一步做什么？

## 回答

1. HTTPS + 国内多节点测域名
2. 看全国红 / 单省红 / 单网红
3. 对比 HTTP vs HTTPS（证书问题）
4. 必要时测全球节点（跨境）

| 结果 | 方向 |
|------|------|
| 全国红 | 服务器、证书、443 防火墙 |
| 单省红 | 区域线路、DNS 缓存 |
| 仅移动红 | 三网/移动优化 |

定性后再查 Nginx 日志，别盲目重启。
