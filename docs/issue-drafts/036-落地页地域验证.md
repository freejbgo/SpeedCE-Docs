# Issue 标题

问：广告投放省份和实际可达性怎么交叉验证？

---

# Issue 正文

## 问题

广告投某省，落地页在该省打不开，转化差。

## 回答

用多节点地图看**投放省份**是否绿。若投放省红、邻省绿 → 查落地页 CDN、备案、运营商。投放前对落地页 URL 做分省测速存档。
