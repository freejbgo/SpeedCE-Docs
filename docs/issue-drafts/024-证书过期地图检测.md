# Issue 正文（复制到描述框）

## 摘要

证书静默过期时，运维本地可能仍正常。SpeedCE HTTPS + 全国节点，若大面积异常且 HTTP 正常，优先续签证书并检查 SAN 是否覆盖所有子域。

续签后 5 分钟复测，地图转绿再宣布恢复。建议每月 1 号 HTTPS 巡检 + 到期提醒。免费工具：https://speedce.com/?lang=zh-CN。
