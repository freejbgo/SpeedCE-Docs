# Issue 标题

问：为什么在线测速工具不能测 192.168.x.x？

---

# Issue 正文

## 问题

想测内网服务，输入内网 IP 被拒绝。是工具 bug 吗？

## 回答

不是 bug，是**安全设计**。公网测速平台若允许测内网地址，可能被滥用扫描他人内网（SSRF 类风险）。

正确做法：

- 内网服务用内网 curl / 内网监控
- 外网可达性用公网域名或多节点外网工具
- 需要「外网视角」时测公网 IP 或已解析的域名

内外网结论不能互相替代。
