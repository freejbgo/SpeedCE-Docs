# Issue 标题

问：远程办公时，不连公司 VPN 怎么验证全国状态？

---

# Issue 正文

## 问题

居家无法代表用户网络，也不想每次连 VPN。

## 回答

浏览器打开多节点测速即可从外网验收，无需跳板机。分布式团队可约定：争议时以**外网多节点截图**为准，减少「我本地 OK」。
