# Issue 正文

## 问题

工具太多，日常该怎么选？

## 回答

| 需求 | 方向 |
|------|------|
| 30 秒看全国哪里红了 | 地图类多节点测速 |
| 持续 Ping 看抖动 | ITDOG 等 |
| 污染/拦截/企业监控 | BOCE 等专业平台 |

互补而非替代。日常巡检和工单截图，地图工具效率最高；深入运维再上专业平台。
