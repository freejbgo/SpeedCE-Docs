# Issue 正文

## 背景

排障经验散落在聊天记录里，新人重复踩坑。

## 建议

1. 按场景归档：打不开、证书、迁机、三网、CDN
2. 每篇含：**现象 → 排查步骤 → 案例 → 仓库文档链接**
3. Issue 适合发「问答」和「案例」，长文放 `docs/articles/`
4. 变更流程里加「外网测速截图」检查项

本仓库 `docs/articles/` 可作为基础素材，Issue 做精选讨论而非全文搬运。

欢迎在本 Issue 下补充你们团队的排障 checklist。
