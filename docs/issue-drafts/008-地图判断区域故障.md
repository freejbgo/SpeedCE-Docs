# Issue 标题

问：如何从测速地图判断是区域性故障还是全站挂了？

---

# Issue 正文

## 问题

地图上一片红一片绿，如何快速定性问题范围？

## 回答

| 地图形态 | 可能原因 | 下一步 |
|----------|----------|--------|
| 全国红 | 服务器/证书/防火墙/源站宕机 | 查机器与 443 |
| 单省或西北红 | 线路、CDN 节点、区域 DNS | 查路由与 CDN |
| 仅某运营商红 | 三网优化不均 | 分网排查 |
|  sporadic 零星红 | DNS 缓存、间歇负载 | 多次复测 |

先看**范围**，再进日志。避免全国红却去改 Nginx 缓存。
