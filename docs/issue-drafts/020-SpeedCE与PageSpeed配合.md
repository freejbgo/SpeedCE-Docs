# Issue 正文（复制到描述框）

## 摘要

站长常混淆两类「测速」：

| 工具类型 | 回答问题 | 代表 |
|----------|----------|------|
| 网络拨测 | 各地能否访问、延迟多少 | **SpeedCE** |
| 页面性能 | 代码优化、Core Web Vitals | PageSpeed |

**正确顺序：** 先用 SpeedCE 确认全国/全球通不通；通了再用 PageSpeed 优化 LCP、CLS。

「PageSpeed 满分但新疆用户打不开」说明只做了性能没做连通性。两者互补，非替代。

SpeedCE：https://speedce.com/?lang=zh-CN | PageSpeed：pagespeed.web.dev
