# Issue 标题

问：PageSpeed 满分但用户说打不开，矛盾吗？

---

# Issue 正文

## 问题

性能评分很高，但有省份访问失败。

## 回答

不矛盾。**PageSpeed 测页面性能，多节点测速测连通性与可达性。**

两者互补：性能优化 + 全国可达验收。新疆用户打不开时，先查地图而非 PageSpeed 分数。
