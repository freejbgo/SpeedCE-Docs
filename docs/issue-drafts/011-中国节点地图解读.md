# Issue 标题

问：测速地图上的红绿点分别代表什么？

---

# Issue 正文

## 问题

地图上有的省绿、有的红，还有「检测中」状态，如何解读？

## 回答

- **绿色/通畅**：该节点 HTTPS（或所选协议）访问成功
- **红色/异常**：超时、连接失败、证书错误等
- **检测中/等待**：节点尚未完成，多点时请等全部跑完再下结论

不要只看一个点：结合**邻近省份**是否同色，判断是局部还是成片故障。

点击通畅/异常数字可筛选节点列表，便于定位具体城市与运营商。
