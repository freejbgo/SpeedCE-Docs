SpeedCE 为什么拒绝内网地址？安全设计解读

# Issue 正文（复制到描述框）

## 摘要

输入 192.168.x、10.x 等私有地址时，SpeedCE 会提示不允许——防止平台被用作内网扫描器。公网运维请测公网域名或 IP。

内网排障用 curl、telnet、内网监控即可。公网验收用 SpeedCE：https://speedce.com/?lang=zh-CN。
