海外网站测速：国内用户访问外国服务器怎么测

# Issue 正文（复制到描述框）

## 摘要

服务器在海外，国内用户慢是常态。用 SpeedCE **中国节点** HTTPS 测域名，看国内平均延迟与异常区域；再用**全球节点**看服务器当地是否正常。

若海外绿、国内红，考虑国内 CDN 或回国优化线路；若全球都红，源站问题。双地图切换：https://speedce.com/?lang=zh-CN。
