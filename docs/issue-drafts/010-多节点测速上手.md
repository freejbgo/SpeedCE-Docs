# Issue 正文

## 问题

没用过地图类测速，想五分钟完成第一次验收，步骤是什么？

## 回答

1. 选 **HTTPS**（最接近用户访问）
2. 选 **中国节点**（看国内用户）
3. 输入**主域名**（不是内网 IP）
4. 看地图红绿分布 + 通畅率
5. 需要时筛选电信/联通/移动分别查看

有海外用户再切全球节点补测。结果建议截图带时间戳存档。
