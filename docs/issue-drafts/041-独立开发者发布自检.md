# Issue 标题

问：独立开发者上线 Product Hunt 前，网络要检查什么？

---

# Issue 正文

## 问题

Landing 页和 API 准备发布，怕第一波流量遇到区域性故障。

## 回答

5 分钟自检：HTTPS 测主域；中国节点看国内；全球节点看目标市场；关键 API 子域单独测。第一印象来自可访问性。截图备查，发布日可对照。
