# Issue 正文

## 问题

游戏服延迟高，是否立刻优化游戏逻辑？

## 回答

先定性：**全网卡还是某省卡？**

PING/HTTPS 测入口域名或 IP，看国内区域分布。仅东北红查机房路由；全国红查负载或攻击。

网络层用地图定性后，再进服日志，避免盲目改代码。
