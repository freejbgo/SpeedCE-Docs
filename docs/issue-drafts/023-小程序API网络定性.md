# Issue 标题

问：小程序报错，先查代码还是查 API 域名可达性？

---

# Issue 正文

## 问题

小程序请求失败，团队争论是前端还是后端。

## 回答

先 HTTPS 测 **API 域名**全国节点：

- 大面积红 → 优先网络/证书/Nginx，而非急着发版小程序
- 全绿 → 再查业务代码、域名白名单、微信侧限制

网络层定性通常只需几分钟。
