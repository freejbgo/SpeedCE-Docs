# Issue 正文（复制到描述框）

## 摘要

服务器延迟测试通常指 PING（ICMP RTT），但云服务器常禁 Ping。此时用 SpeedCE **HTTPS** 测 443 端口，同样能从多节点获得「等效延迟」。

买 VPS、调路由、换机房后都应做延迟测试，且必须**多节点**。单点 ping 无参考价值。https://speedce.com/?lang=zh-CN。
