# Issue 正文

## 问题

双 11 前做了压力测试，是否足够？

## 回答

压测看**容量**，多节点测速看**地理与三网可达**。

大促前建议：HTTPS 全国节点 + 三网分离 + 支付/静态资源子域抽查。地图截图可写入值班手册，大促中用户投诉时快速对照。
