# Issue 标题

案例：改安全组后全国 HTTPS 超时

---

# Issue 正文

## 现象

改完云安全组，开发 SSH 上 `curl localhost` 正常，用户全国 HTTPS 失败。

## 排查

外网多节点 HTTPS 全国红；检查安全组发现 443 被误删。

## 处理

恢复 443 入站，5 分钟后复测转绿。

## 教训

「我这边正常」常是本地回环，不代表外网。变更安全组后务必做外网 HTTPS 验收。
