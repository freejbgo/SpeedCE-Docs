# Issue 正文

## 问题

开课在即，如何快速确认「各地学生能进教室」？

## 回答

HTTPS 测课堂域名/直播页，看**通畅率**而非平均延迟。电信、联通、移动各看一眼，重点省份（学生来源）是否全绿。异常则切备用线路或 CDN，而非临时改课件。
