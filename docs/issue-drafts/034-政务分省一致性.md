# Issue 标题

问：政务网站如何验证「各省都能访问」？

---

# Issue 正文

## 问题

上级要求分省访问一致性，如何自证？

## 回答

HTTPS + 中国全部省份节点测主域，导出地图与通畅率。对比三网结果，某省持续红则查区域链路或解析。建议月度固定巡检存档。
