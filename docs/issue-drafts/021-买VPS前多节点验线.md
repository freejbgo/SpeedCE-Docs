## 摘要

VPS 商家文案里的「CN2 GIA」「精品网」「三网直连」，需要第三方数据验证。推荐流程：下单拿到 IP → SpeedCE PING + 中国节点 → 三网筛选各截图 → 全球节点补测（若面向海外）。

若电信联通延迟 30–80ms、移动 150ms+，说明偏电信联通优化。若全国 sporadic 红，查安全组与防火墙。退款期内不满意即换，地图证据可发帖维权。

工具：https://speedce.com/?lang=zh-CN。买 VPS 不验线，等于盲盒。
