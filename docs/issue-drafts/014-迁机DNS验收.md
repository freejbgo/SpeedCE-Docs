# Issue 标题

问：迁机改完 DNS，多久能确认全国生效？

---

# Issue 正文

## 问题

服务器已迁完，后台能访问，如何确认全国用户都解析到新 IP？

## 回答

1. 迁机前把 TTL 调低（如 300s）
2. 改 DNS 后测**域名**（不是只 ping 新 IP）
3. 每 10–15 分钟测一次，看异常点是否减少
4. 24 小时后仍固定某省红 → 查 DNS 配置，不是继续等

全国仍红但 IP 已变 → 查新服务器 Web/证书/安全组，不是 DNS 问题。
