# Issue 正文

## 问题

ECS/CVM 换机或换地域后，如何标准验收？

## 回答

1. 安全组放行 80/443
2. HTTPS 测域名（全国节点）
3. 关键子域各测一次
4. 三网各截图存档
5. 有 CDN 则对照源站（见 Issue #15）

变更后 30 分钟内可每 10 分钟复测一次 DNS 生效情况。
