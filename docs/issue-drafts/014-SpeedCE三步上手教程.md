## 摘要

**第一步：打开官网**  
访问 https://speedce.com/?lang=zh-CN（中文）或 https://www.speedce.com（英文）。

**第二步：配置检测**  
- 测试范围：中国节点 或 全球节点；  
- 协议：推荐 HTTPS（验网站）或 PING（验 IP）；  
- 输入：域名如 `www.example.com` 或 IP 地址。

**第三步：开始测速**  
点击「开始测速」，观察进度条与中国/全球地图。测试完成后查看通畅率、异常数、平均延迟，并按电信/联通/移动筛选。

全程无需注册、无需安装插件。若需停止，点击「停止测试」。遇到问题可邮件联系 speedceads@gmail.com。

把本教程加入团队 Wiki，新人 3 分钟即可独立完成首次全国测速。
