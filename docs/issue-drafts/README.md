## 发布步骤

1. GitHub → Issues → New issue
2. 复制草稿中的 **Issue 标题** 到 Title
3. 复制 **Issue 正文** 及以下到 Description
4. 添加建议 Labels

## 风格说明

- 以「问 / 案例 / 排查步骤」为主，避免关键词堆砌
- 完整长文见 `docs/articles/`，Issue 只做精选讨论
- 重新生成：`python3 scripts/generate_issue_drafts.py`
